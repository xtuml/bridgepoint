What’s New in BridgePoint 4.0.3

Release Notes

 

New Products, New Licensing

This release includes the open source xtUML Editor from xtUML.org.  The editor is no longer restricted.  Model execution and translation capabilities remain proprietary tools.  The product offerings from Mentor Graphics have changed to reflect this transition.

BridgePoint users who wish to upgrade to this release must contact Mentor Graphics to receive a new license file.
