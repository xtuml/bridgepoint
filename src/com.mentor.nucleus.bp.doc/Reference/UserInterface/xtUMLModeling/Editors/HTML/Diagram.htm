Diagram Editor

Diagram editors provide a graphical view in which to display and edit model data.    Each diagram has the same look and feel and behaves in the same way.  Below is the Domain Package Diagram for the Microwave Oven sample model.

All diagram editors have a toolbar on which there are tools.  Tools are available for the most common operations – other operations are available from context menus.  Some of the tools are common to all diagrams, some common to several, and some specific to the diagram itself.

The following figure shows the toolbar for the Domain Package Diagram. 

The left-most tools are common to all diagrams and provide zoom in, zoom out, zoom all, and zoom selected functionality.  The arrow is the selection tool, and is used to select model elements on the diagram.  The right-most tools put the diagram in “input mode” allowing model elements to be drawn on the diagram.

Hovering over a tool shows the tool tip associated with the tool.  The tool tip corresponds to the button name.  The following is a list of the operations available on all canvases:

·         Symbol display, drawing, selection, movement, and resize

·         Selection highlighting

·         Zooming, scrolling and panning

·         Context menu activation

·         Scaling

·         Waypoint manipulation

·         Diagram printing

Diagram editors provide a way to navigate the model (using zooming, panning, and scrolling) to gain access to the different model elements for inspection and edit.  Pressing the space bar on a diagram changes the cursor to a hand symbol in order to grab and move the canvas to the desired location.  Specific toolbars and operations for each of the diagrams are discussed in further detail under Diagram Toolbars.

Navigating from a Class to an Instance or State Machine Diagram

You can access the Instance or State Machine Diagram for a class from the Class Diagram. The appropriate icon is displayed in the upper left-hand corner of the class symbol as shown in the image below. 

To access the Instance or State Machine Diagram, double-click the icon.

If the class has both an Instance State Machine and a Class State Machine, both icons appear in the upper left-hand corner of the class symbol.

You can also right-click the appropriate State Machine icon to access a context menu. This menu is identical to the context menu available if you right-click the State Machine icon in the Model Explorer view or Outline view.

Snap Grid

All diagram editors have snap grid enabled to help you align symbols and draw straight lines. Snap grid is an invisible grid that defines the valid set of snap points used to position shapes and connectors. The snap points are the locations on the drawing canvas where a shape or connector can start or end. Snap grid is always enabled except where connectors attach to other connectors.

Snap grid helps you keep your shapes and connectors aligned in the editor when drawing new shapes/connectors or resizing existing ones. The only exception to this is when you resize a shape originally drawn in a release of BridgePoint prior to version 7.0.5. In this case, the fixed part of the shape remains in its original position which may not coincide with the snap points.
