= What's New in BridgePoint 7.8

== Release Highlights
* Verifier updates
  ** Change Verifier to read the project `.classpath` file for realized Java code
  ** Support SDTs in realized Verifier code
  ** Add CME to execute instance-based operations in a Verifier session
  ** Add `STRING` EE implementation to Verifier
  ** Support reference array parameters in realized EEs for Verifier
  ** Various bug fixes
* Textual xtUML support
  ** Support package references
  ** Various bug fixes
* Model load process
  ** Enhancements to stability and efficiency of model load
* Bug fixes
