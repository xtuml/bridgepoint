3.5. User Defined Types

A user defined type is a reference to a type that has already been declared. Types defined in the current domain can be named using just their type name. Types from other domains can be accessed by using a scoped name. Only public types from other domains are accessible. Use of this qualified form, where the type is in the enclosing domain, is allowed.

[51] user defined type = scoped name ;  

A compile-time error occurs if the scoped name does not name a type that is visible to the enclosing domain.
