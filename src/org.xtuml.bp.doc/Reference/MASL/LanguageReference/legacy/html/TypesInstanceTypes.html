3.4. Instance Types

An instance type is a type that refers to an instance of a given object.

[50] instance type = 'instance', 'of', scoped name ;  

A compile-time error occurs if the scoped name does not name an object that is visible to the enclosing domain.

The features of an instance type are given in Table 3.9, “Instance Type Features”, where i and i2 represent instances on which the feature is applied and O represents the object.

Table 3.9. Instance Type Features
Feature Description
= Returns true if i is equal to i2.
/= Returns true if i is not equal to i2.
-> Returns either the instance or bag of instances that are related to i using the specified relationship.
with_-> Returns either the associative instance that models a specific relationship between i and i2.
find Returns the set that contains all the instances of O that satisfy a condition.
find_all Returns a set containing all the instances of O.
find_one Returns an arbitrary instance of O that satisfies the specified condition.
find_only Returns the only instance of O that satisfies the specified condition.
