12.2. Sequence of Statements

A sequence of statements is a sequence of one or more statements.

[150] sequence of statements = statement, {statement} ;  

A sequence of statements is executed by executing each of the statements in order from first to last (left to right). If all of these statements complete normally, then the sequence of statements completes normally. If any of these statements complete abruptly for any reason, then the sequence of statements completes abruptly for the same reasons.
