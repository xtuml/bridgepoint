12.15. The Exit Statement

An exit statement transfers control out of an enclosing while statement or for statement.

[174] exit statement = 'exit', [('when', condition)], ';' ;  

An exit statement is executed by evaluating the optional condition. If the condition evaluates to true or if no condition is present, then the exit statement transfers control to the innermost enclosing while statement or for statement; this statement, which is called the break target, then immediately completes normally.

A compile-time error occurs if there is no while statement or for statement; that encloses the exit statement.
