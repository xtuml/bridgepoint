External Entity Package

External Entity packages organize the external entities that an xtUML model needs to interact with.  EE packages consist of external entities and other EE packages.

External Entity

An external entity represents something outside of the xtUML model.   EE’s can be other xtUML models, or pieces of code to which your xtUML models need to interface.

Bridge Operation

An operation defined on an external entity.  Bridge operations can take parameters and return values.  

Both external entities and bridge operations can be “mapped” by the marking facilities of the Model Compiler.  Some common mappings include:

·         Mapping bridge operations to functions in another xtUML model

·         Mapping bridge operations to operations defined in header files.
